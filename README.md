# How to Use

## Clone the repository

```sh
 
git clone https://github.com/eduahcb/dtmoney

cd dtmoney

```

### Install Dependencies

```sh
  yarn install
```

### Run the project

```sh
 yarn dev
```

### Run the testes

```sh
 yarn test
```

### Run the Test Coverage

```sh
 yarn coverage
```
